# QQbot---airi
This project contains some plugin source code for the Airi bot, which is used to connect to Nonebot via the standardized interface provided by Napcat. 

The source code is here, and hopefully it will be helpful to everyone.

这个项目里面有一些关于airi机器人的一些插件源码，是配合napcat提供的标准化接口连接nonebot使用的，源码就放在这里了，希望能对大家有些作用
